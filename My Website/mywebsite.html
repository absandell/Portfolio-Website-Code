Portfolio Website

HTML5 | CSS | JavaScript | PHP | Front-End | Back-End | Web Development


My portfolio website was a project undertaken with the purpose of teaching myself front-end web development. I tackled the project by first learning HTML5, then CSS, followed by PHP and JavaScript for any additional functionality. The animated background is built using an open-source github project: "particles.js" which I have slightly modified to fit my needs.

This website will be continually updated as I complete more projects that I wish to showcase. I will also be periodically innovating the design and looking to add more functionality and features with the purpose of increasing the scope of the website.
